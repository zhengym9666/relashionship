# Relationship Conflict Navigator - MVP需求文档

## 1. 产品概述

### 1.1 产品定位
Relationship Conflict Navigator 是一个专注于帮助用户解决亲密关系中敏感话题沟通障碍的工具型网站。通过提供结构化的对话模板，降低用户开启困难对话的焦虑感，促进健康的关系沟通。

### 1.2 核心用户痛点
- 不知道如何与伴侣开启敏感话题的对话
- 担心不当的表达方式引发更大冲突
- 缺乏结构化的沟通框架和参考模板

### 1.3 MVP目标
- 实现单一核心功能：输入冲突类型，生成对话模板
- 验证产品价值和用户需求
- 为后续功能迭代收集用户反馈

## 2. 核心功能

### 2.1 冲突类型选择
- 提供预设的关系冲突类型列表
- 支持用户搜索冲突类型
- 覆盖主要敏感话题：出轨、卫生习惯、信任背叛、财务问题、家庭矛盾等

### 2.2 对话模板生成
- 根据用户选择的冲突类型，自动生成3-5个结构化的对话模板
- 每个模板包含：开场句、核心表达、情感反馈、解决方案引导
- 模板语言温和、非指责性，注重共情和理解

### 2.3 模板展示与复制
- 清晰展示所有生成的对话模板
- 支持一键复制单个模板内容
- 提供模板使用建议



## 3. 用户流程

```
用户访问网站 → 浏览/搜索冲突类型 → 选择特定冲突类型 → 系统生成对话模板 → 用户查看/复制模板
```

### 3.1 详细流程
1. **首页访问**：用户打开网站，看到产品介绍和冲突类型选择界面
2. **冲突选择**：用户从列表中选择或搜索特定的冲突类型
3. **模板生成**：系统根据选择的冲突类型，动态生成相应的对话模板
4. **模板使用**：用户查看模板内容，选择合适的模板进行复制和使用

## 5. 界面设计

### 5.1 设计原则
- 简洁明了：避免不必要的视觉元素，专注核心功能
- 温暖友好：使用柔和的色调和亲切的语言，减少用户焦虑
- 响应式设计：适配桌面和移动设备

### 5.2 页面布局

#### 首页 (/) 
- **顶部区域**：
  - 产品名称：Relationship Conflict Navigator
  - 简短标语："让困难的对话变得更简单"

- **主要内容区**：
  - 冲突类型选择器：下拉菜单/搜索框组合
  - 冲突类型列表：显示所有可用的冲突类型
  - 快速提示：简短的使用指南

- **结果展示区**：
  - 模板标题："为您生成的对话模板"
  - 模板卡片：每个模板一个卡片，包含完整的对话结构
  - 复制按钮：每个模板卡片配备复制功能

- **页脚区域**：
  - 简单的产品介绍
  - 联系信息/反馈入口

### 5.3 色彩方案
- 主色调：柔和的蓝色 (#3B82F6) - 代表信任和安全感
- 辅助色：温暖的粉色 (#EC4899) - 代表情感和连接
- 中性色：白色、浅灰色 - 保持界面简洁清晰

### 5.4 字体选择
- 主字体：Inter - 现代、易读的无衬线字体
- 字体大小：
  - 标题：24-32px
  - 正文：16-18px
  - 辅助文字：14px
